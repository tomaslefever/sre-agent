# AgentX SRE — 90-Second Demo Script

---

**[0:00 - 0:08] INTRO**
*Screen: Sidebar with AgentX SRE logo*
    
> "AgentX SRE is an autonomous SRE agent that diagnoses incidents, analyzes source code, and generates automatic fixes with Pull Requests on GitHub."

---

**[0:08 - 0:25] REPORT AN INCIDENT**
*Screen: Diagnostic Center — paste image with Ctrl+V and type message*

> "We report an incident by pasting a screenshot directly from the clipboard. The agent analyzes the image with GPT-4o Vision, exhaustively searches the indexed codebase, and generates a detailed technical diagnosis."

---

**[0:25 - 0:35] TICKET CREATED**
*Screen: Linked ticket banner appears in chat — click "View Ticket"*

> "The agent automatically creates a ticket with the report, assigns it to a technician, and links it to this conversation. We access the ticket with a single click."

---

**[0:35 - 0:52] GENERATE ACTION PLAN**
*Screen: Ticket detail — click "Generate Action Plan" — result with files, findings, and plan*

> "From the ticket, we generate an AI-powered action plan. The system searches the vector database, identifies affected files, lists concrete findings with code offsets, and proposes specific steps to resolve the issue."

---

**[0:52 - 1:05] EXECUTE PLAN AND SEND PR**
*Screen: Inside the plan — click "Execute Plan" — then "Send PR" — history shows activity*

> "We execute the plan: the agent generates corrected code and pushes it to a descriptive branch on GitHub. When ready, we manually send the Pull Request. The ticket moves to validation status and plans are locked until the PR is reviewed."

---

**[1:05 - 1:18] KANBAN BOARD + HISTORY**
*Screen: Return to board — show 5 columns — ticket in Validation column — right sidebar with history*

> "The Kanban board shows the full lifecycle: open, in progress, review, validation, and resolved. Every action is logged in the ticket history, including errors if any occur."

---

**[1:18 - 1:30] KNOWLEDGE BASE + SESSIONS**
*Screen: Knowledge Base — linked repository — back to sidebar with sessions*

> "The knowledge base syncs GitHub repositories and indexes the code into a vector database. Conversations are saved as reusable sessions, always linked to their tickets."

---

**[1:30] CLOSING**
*Screen: General dashboard view*

> "AgentX SRE: from incident to Pull Request, without leaving the platform."

---

### Production Notes
- **Pace**: ~150 words/minute (natural narration speed)
- **Transitions**: direct cuts, no long animations
- **Visual focus**: follow mouse cursor on each key action
- **Audio**: voiceover + low ambient music
